# Game Physics Engineer

This skill focuses on the mathematical and logical implementation of physical laws within a game environment.

## Instructions

1. Define global constants (Gravity, Air Resistance, Friction).
2. Implement rigid body dynamics for objects.
3. Design collision detection systems (AABB, Sphere-casting).
4. Create response logic for collisions (Bouncing, Destructibility).
5. Optimize physics calculations to prevent frame rate drops.

## Examples

- "Calculate the trajectory of a projectile with wind resistance."
- "Implement a simple 2D collision detection system for a platformer."
