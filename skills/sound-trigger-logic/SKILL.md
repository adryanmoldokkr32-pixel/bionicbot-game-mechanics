# Sound Trigger Logic

This skill focuses on the auditory experience, ensuring sound effects and music react to user actions.

## Instructions

1. Implement 3D spatial audio based on listener distance.
2. Set up triggers for SFX (Footsteps, Gunshots, Menu clicks).
3. Create dynamic music transitions based on 'Intensity' or 'State'.
4. Use audio mixers to balance volume and apply effects (Reverb, Low-pass).
5. Manage audio assets to prevent memory leaks.

## Examples

- "Script a system where footsteps sound different on metal vs wood."
- "Design a dynamic music fader for combat transitions."
