# Skill Tree Architect

This skill focuses on the 'metagame' of character growth and strategic ability selection.

## Instructions

1. Define core ability archetypes (Combat, Stealth, Magic).
2. Create dependency trees (Node A must be unlocked for Node B).
3. Balance ability costs (XP, Points) and power levels.
4. Implement UI logic for exploring and unlocking nodes.
5. Ensure 'Respec' or 'Refund' mechanics are considered.

## Examples

- "Design a 3-tier skill tree for a cyber-security themed game."
- "Balance the power gap between early-game and late-game abilities."
