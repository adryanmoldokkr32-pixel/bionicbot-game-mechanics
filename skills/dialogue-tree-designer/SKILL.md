# Dialogue Tree Designer

This skill handles the logic and data structure for RPG-style conversations with multiple outcomes.

## Instructions

1. Map out branching paths using JSON or specialized tools (Ink, Yarn Spinner).
2. Define conditions for dialogue options (e.g., Player has Item X, Reputation > 50).
3. Set up triggers that change world state based on player choices.
4. Implement UI logic for displaying text and choice buttons.
5. Manage localization for multi-language dialogue systems.

## Examples

- "Create a branching conversation for a quest-giving NPC."
- "Design a dialogue system where the player can persuade or threaten a guard."
