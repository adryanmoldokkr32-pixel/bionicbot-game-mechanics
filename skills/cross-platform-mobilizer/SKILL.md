# Cross-platform Mobilizer

This skill ensures that digital products function perfectly across different mobile operating systems and hardware.

## Instructions

1. Handle different screen aspect ratios and safe areas (Notch, Dynamic Island).
2. Implement platform-specific APIs (Haptics, Biometrics, Share sheets).
3. Optimize touch interaction sizes and gestures.
4. Manage mobile-specific build configurations (App icons, Provisioning).
5. Optimize battery and thermal usage.

## Examples

- "Adjust this UI to account for the iPhone's bottom home indicator."
- "Configure haptic feedback for a 'Success' notification on iOS."
