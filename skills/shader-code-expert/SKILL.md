# Shader Code Expert

This skill enables bionicbot to create visual effects directly on the GPU, such as water ripples, glows, or post-processing.

## Instructions

1. Define vertex and fragment shader logic.
2. Implement mathematical effects (Sine waves, Noise, Distortions).
3. Pass uniforms (Time, Resolution, Mouse position) from CPU to GPU.
4. Optimize shader math to maintain high FPS.
5. Design post-processing filters (Blur, Grain, Color Grading).

## Examples

- "Write a GLSL fragment shader for a realistic water surface."
- "Create a 'pixelate' effect shader for a retro-style game."
