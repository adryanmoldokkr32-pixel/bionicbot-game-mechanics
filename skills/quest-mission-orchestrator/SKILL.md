# Quest & Mission Orchestrator

This skill handles the flow of objectives, rewards, and story progression in a game.

## Instructions

1. Define mission types (Fetch, Kill, Escort, Reach).
2. Create clear objective markers and tracking.
3. Design branching dialogue or choice-based mission outcomes.
4. Set up reward systems (XP, Gold, Items).
5. Manage quest logs and active/completed states.

## Examples

- "Design a multi-stage quest for a fantasy RPG."
- "Script the mission logic for a tutorial level in a racing game."
