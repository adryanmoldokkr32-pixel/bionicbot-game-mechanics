# Simulated Economy Loop

This skill focuses on 'Risk-Free Innovation', running 1 million Monte Carlo simulations of an economy or business model to ensure stability and profit.

## Instructions
1. Define agents (Users), resources (Money), and rules (Taxes, Prices).
2. Inject 'Chaos' events (Crashes, Hype, Competition).
3. Monitor for Hyper-inflation, Monopoly, or System Collapse.
4. Optimize rules to ensure long-term ecosystem health.

## Examples
- "Run a 1-million user simulation of our '100x Growth' token model to find the crash point."
