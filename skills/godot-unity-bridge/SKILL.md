# Godot/Unity Bridge

This skill focuses on the specific APIs and structures of the world's most popular game engines.

## Instructions

1. Map abstract game logic to Engine nodes/components.
2. Use GDScript (Godot) or C# (Unity) for scripting.
3. Configure Scene/Prefab hierarchies for modularity.
4. Set up signals (Godot) or events (Unity) for communication.
5. Optimize build settings for different platforms.

## Examples

- "Translate this character movement logic into a Godot GDScript."
- "Show how to set up a Prefab for an enemy in Unity."
