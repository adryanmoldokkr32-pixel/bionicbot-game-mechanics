# Omni-Engine Porting

This skill enables bionicbot to act as a universal game developer, porting projects between Godot, Unity, Unreal, or WebGL with zero manual recoding.

## Instructions
1. Analyze the source project structure and script logic.
2. Use AI transpilers to convert code to the target language (C#, GDScript, C++).
3. Map shaders, materials, and physics settings to target engine equivalents.
4. Automatically generate build files for iOS, Android, and Desktop.

## Examples
- "Port our 2D platformer from Godot to Unity for a special console release."
