# Inventory System Logic

This skill focuses on the data structure and UI flow for handling in-game items.

## Instructions

1. Define item schemas (ID, Name, Type, Weight, Rarity).
2. Implement grid-based or list-based storage logic.
3. Design 'Equip' and 'Use' mechanics with stat modifiers.
4. Create loot drop tables based on percentage chances.
5. Handle persistence (saving/loading inventory state).

## Examples

- "Structure a database schema for an RPG inventory system."
- "Create the logic for a crafting system that combines two items."
