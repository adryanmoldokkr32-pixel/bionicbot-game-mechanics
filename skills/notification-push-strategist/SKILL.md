# Notification Push Strategist

This skill focuses on keeping users active and informed without being intrusive or annoying.

## Instructions

1. Set up push providers (Firebase Cloud Messaging, OneSignal).
2. Request permissions at the right psychological moment.
3. Design 'Rich Notifications' with images and interactive buttons.
4. Implement deep-linking to specific app sections.
5. Use 'Silent Notifications' for background data syncing.

## Examples

- "Draft a strategy for onboarding push notifications."
- "Implement a button in the notification to 'Reply' to a message directly."
