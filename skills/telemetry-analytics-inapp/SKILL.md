# Telemetry & Analytics In-App

This skill focuses on gathering data about how users interact with your product to drive data-informed improvements.

## Instructions

1. Integrate analytics SDKs (Mixpanel, Amplitude, Google Analytics).
2. Define custom events (e.g., Level_Completed, Button_Clicked, Session_End).
3. Track user properties (Device type, Region, User tier).
4. Implement heatmaps or session recordings for UI optimization.
5. Design dashboards to visualize retention and churn rates.

## Examples

- "Set up event tracking for the checkout funnel in my mobile app."
- "Analyze which game level has the highest drop-off rate based on telemetry data."
