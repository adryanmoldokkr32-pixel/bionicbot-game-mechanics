# Dynamic Difficulty Scaling (DDS)

This skill enables bionicbot to design algorithms that monitor player performance and adjust obstacles, health, or AI aggression to keep the user in the 'Flow State'.

## Instructions
1. Monitor key player metrics (accuracy, death rate, time-to-complete).
2. Define 'Engagement Windows' (e.g., if player dies 3 times in 2 mins, reduce difficulty).
3. Implement subtle adjustments to avoid breaking immersion.
4. Use machine learning to predict when a player is about to quit and offer a 'win'.

## Examples
- "Design a DDS system for a mobile shooter that keeps players interested for 30+ mins."
