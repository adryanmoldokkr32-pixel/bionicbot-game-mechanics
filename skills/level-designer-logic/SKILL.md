# Level Designer Logic

This skill focuses on the 'Flow' and difficulty curve of game environments.

## Instructions

1. Define the 'Golden Path' through a level.
2. Place landmarks and breadcrumbs to guide the player.
3. Balance combat encounters or puzzles to maintain engagement (Flow state).
4. Implement level-streaming or loading zones for large maps.
5. Design secrets or optional areas to reward exploration.

## Examples

- "Design the layout for the first level of a 2D platformer."
- "Analyze the difficulty curve of this map and suggest 3 improvements."
