# NPC AI Behavior

This skill enables bionicbot to design complex decision-making systems for game characters (Finite State Machines, Behavior Trees).

## Instructions

1. Define NPC states (Idle, Patrolling, Chasing, Attacking).
2. Create transition rules between states.
3. Implement pathfinding logic (A* algorithm).
4. Design sensory systems (Line of sight, hearing range).
5. Balance NPC difficulty to ensure a fair but challenging experience.

## Examples

- "Draft a behavior tree for a stealth game guard."
- "Script an NPC that follows the player but maintains a safe distance."
