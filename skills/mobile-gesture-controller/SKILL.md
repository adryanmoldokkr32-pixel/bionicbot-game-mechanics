# Mobile Gesture Controller

This skill focuses on making mobile interactions feel fluid, natural, and responsive.

## Instructions

1. Implement basic gestures (Single-tap, Double-tap, Long-press).
2. Design 'Swipe-to-delete' or 'Pull-to-refresh' patterns.
3. Implement multi-touch support for zooming and rotating.
4. Set threshold values to prevent accidental triggers.
5. Use physics-based 'Inertia' for scrolling and swiping.

## Examples

- "Implement a pinch-to-zoom gesture for an image gallery."
- "Add a smooth swipe-left interaction to archive a list item."
