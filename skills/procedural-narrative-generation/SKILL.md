# Procedural Narrative Generation

This skill focuses on 'Infinite Replayability' by creating stories that evolve differently every time.

## Instructions
1. Define 'Narrative Nodes' and their connection requirements.
2. Use LLMs to generate dialogue and plot twists based on the player's 'Social Graph'.
3. Ensure logical consistency across branching paths using a 'Master Lore' validator.
4. Automatically generate quest objectives that fit the current state of the world.

## Examples
- "Create a procedurally generated detective story set in futuristic Bucharest."
