# Procedural World Builder

This skill focuses on using noise functions and rulesets to create unique environments every time a game is played.

## Instructions

1. Select a noise function (Perlin, Simplex) for terrain generation.
2. Define biomes and their environmental rules.
3. Implement 'Seed' based generation for repeatability.
4. Create rules for object placement (Trees, Loot, Enemies).
5. Ensure the generated world is navigable and playable.

## Examples

- "Write an algorithm to generate a random 2D dungeon layout."
- "Design a procedural biome system for a space exploration game."
