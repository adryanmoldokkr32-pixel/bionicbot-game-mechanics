# Multiplayer Sync Master

This skill focuses on the networking layer of games, ensuring all players see the same world state.

## Instructions

1. Implement client-side prediction and server reconciliation.
2. Use interpolation to smooth object movement under lag.
3. Minimize packet size and frequency to save bandwidth.
4. Manage lobby systems and player matchmaking.
5. Protect against cheating through server-side authority.

## Examples

- "Explain how to synchronize player positions in a fast-paced shooter."
- "Design a simple matchmaking flow for a 1v1 chess game."
