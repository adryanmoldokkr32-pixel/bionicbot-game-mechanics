# In-App Purchase Logic

This skill handles the complex and sensitive logic of mobile commerce and store validations.

## Instructions

1. Define product IDs (Consumables, Non-consumables, Subscriptions).
2. Implement secure receipt validation (Server-to-Server).
3. Handle 'Restore Purchases' flow for user convenience.
4. Design a 'Store' UI that highlights current deals and currency.
5. Ensure compliance with App Store and Play Store guidelines.

## Examples

- "Structure the flow for a monthly premium subscription."
- "Implement a secure way to add 'Coins' to a user account after purchase."
